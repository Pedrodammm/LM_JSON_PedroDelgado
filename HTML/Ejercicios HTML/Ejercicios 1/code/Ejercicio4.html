Métodos obsoletos en Java

Métodos antiguos y sus reemplazos:

getYear() ----- LocalDate.getYear()

getMonth() -----LocalDate.getMonthValue()

getDay() -----LocalDate.getDayOfMonth()

toLocaleString() ----- DateTimeFormatter.ofPattern("dd/MM/yyyy").format(...)

Codigo en Java:

            import java.util.Date;
            import java.util.Calendar;
            
            
            public class EjemploObsoleto {
                public static void main(String[] args) {
                    Date fecha = new Date();
                    
                    // Método en desuso
                    int anioObsoleto = fecha.getYear()();
                    
                    // Métodos actuales recomendados
                    Calendar calendario = Calendar.getInstance();
                    int anioActual = calendario.get(Calendar.YEAR);
                    
                    System.out.println("Año con método obsoleto: " + (anioObsoleto + 1900));
                    System.out.println("Año con método recomendado: " + anioActual);
                }
            }
